Um die Plugins zu installieren, sind die in diesem Verzeichnis enthaltenen Ordner
in folgende Verzeichnisse zu kopieren:
- 'assReviewableMultipleChoice' in 'ilias/Customizing/global/plugins/Modules/TestQuestionPool/Questions'
- 'Review' in 'ilias/Customizing/global/plugins/Services/Repository/RepositoryObject'
Anschließend müssen die Plugins in der ILIAS-Administration aktiviert werden.
